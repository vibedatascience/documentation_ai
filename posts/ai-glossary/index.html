AI glossary

Harness
The software wrapped around a model to turn it into an agent. It runs the loop that calls the model, executes its tool calls, assembles context each step, persists state between sessions, and enforces permissions. The model supplies the reasoning and the harness supplies everything else. Source: Anthropic, Effective harnesses for long-running agents.
