Hello, vibelogs: how this site is built

What this is

A running log of AI things I try: agents, tooling, LLM apps, data science workflows. Each post is a walkthrough: what I built, the exact steps, and the gotchas nobody documents.

How it's built

Zero framework. Plain HTML on GitHub Pages. Each post is a folder under posts/ with its own index.html and assets.

vibelogs/
├── index.html
├── about.html
└── posts/
    └── 2026-07-16-hello-vibelogs/
        └── index.html

Adding a post

  1. Copy this post's folder, rename with the new date + slug.
  2. Edit the title, tag, and body.
  3. Add one <a class="entry"> block to the homepage under the right month.
  4. Push. Pages redeploys in ~a minute.
GOTCHA

Post format going forward: What I built → Why → Steps → Gotchas → Links. The gotchas section is the whole point.
