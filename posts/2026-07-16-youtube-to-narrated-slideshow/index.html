Turn long YouTube videos into narrated slideshows

LONG ISLAND CITY, N.Y. -- A 31-minute podcast interview was beaten 3.5 minutes to nil here Wednesday, as a five-stage pipeline turned long YouTube videos into short narrated slideshows in a result that surprised nobody who watched the buildup.

The pipeline, unfancied at kickoff, produced three wins from three: the 31-minute interview cut to 3.5 minutes, a 60-minute football podcast condensed to 9.5, and a 60-minute MIT lecture reduced to an 11.3-minute, 19-chapter explainer. All three finished with designed slides, full-bleed imagery, neural voiceover and slow Ken Burns zooms.

YouTube URL │ ▼ [1] transcript EasySubAPI (or yt-dlp + Whisper) │ ▼ [2] distill LLM condenses to 8-20 slide script │ ▼ [3] slides 1920×1080 cards + real imagery/diagrams │ ▼ [4] voiceover edge-tts, one clip per slide │ ▼ [5] render ffmpeg: zoompan + fades → final.mp4

Transcript wins the midfield battle

Every good move started from the back. For YouTube sources, a transcript API beats downloading the full video for pace and never looked troubled:

import http.client, json

conn = http.client.HTTPSConnection("easysubapi.p.rapidapi.com")
payload = json.dumps({"video_id": "VIDEO_ID"})
headers = {
    "x-rapidapi-key": "YOUR_KEY",
    "x-rapidapi-host": "easysubapi.p.rapidapi.com",
    "Content-Type": "application/json"
}
conn.request("POST", "/api/easysubapi-get-transcript", payload, headers)
data = json.loads(conn.getresponse().read())
frames = data["result"][0]["data"]["frames"]

For non-YouTube sources, the bench option is ffmpeg audio extraction plus local faster-whisper transcription. Reliable, if a step slower.

Rolling captions nearly cause an upset

YouTube's auto-captions repeat every phrase two to four times across frames and arrive riddled with HTML tags. Exact-match dedup alone was chasing shadows. The correct shape: strip tags, unescape, sentence-split, drop consecutive duplicates, dedupe.

import re, html

text = " ".join(re.sub(r"<[^>]+>", "", html.unescape(f["message"])) for f in frames)
sents = re.split(r"(?<=[.!?]) +", text)
sents = [s for i, s in enumerate(sents) if i == 0 or s != sents[i-1]]
clean = " ".join(dict.fromkeys(sents))

From there the LLM runs the distillation: 8-20 slides, each with a headline, 2-3 lines of narration and an image direction. This is where the match is won. Prompt for core ideas and the best verbatim quotes, never a flat summary.

Plain cards flatter to deceive

The first version -- dark text cards, technically sound -- was dire viewing, and the crowd let it hear about it. The turnaround came from three changes at the break:

  • Full-bleed imagery matched to each slide: public domain paintings for a history podcast, stadium photography for football, custom diagrams for a lecture
  • Cinematic grading: darken and desaturate so white text sits on top
  • Programmatic diagrams (PIL/matplotlib) for technical content: network architectures, tactical boards, activation curves. Beats hunting for stock

edge-tts a shrewd signing on a free

Microsoft's neural voices, pip-installable, no fee. One MP3 per slide, so slide timing is driven by audio duration and manual sync never enters the conversation.

pip install edge-tts
edge-tts --voice en-US-GuyNeural --text "$NARRATION" --write-media slide_01.mp3

en-US-GuyNeural for a documentary feel; en-GB-RyanNeural when the source is British.

ffmpeg scores the winner in stoppage time

Each slide pairs its PNG with its MP3, takes a slow zoompan (alternating in and out), crossfades, then concatenates:

ffmpeg -loop 1 -i slide_01.png -i slide_01.mp3 \
  -vf "zoompan=z='min(zoom+0.0008,1.08)':d=125*10:s=1920x1080,fps=25" \
  -c:v libx264 -pix_fmt yuv420p -shortest seg_01.mp4

ffmpeg -f concat -safe 0 -i segments.txt -c copy final.mp4

Talking points

GOTCHA · ROLLING CAPTIONS

Strip tags, unescape, sentence-split, then dedupe. In that order or not at all.

GOTCHA · TLS + EDGE-TTS

Behind an egress proxy with a self-signed cert, edge-tts's websocket TLS verification fails. Bypass verification for that one connection only.

GOTCHA · PLAIN CARDS ARE BORING

Text-on-dark-card slides feel like a settings screen. Imagery per slide is the difference between "delete" and "rewatch".

GOTCHA · ONE SESSION

Ephemeral sandboxes reset. Run the whole pipeline in one go and save intermediates as you go.
